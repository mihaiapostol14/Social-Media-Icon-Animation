# Social Media Icon Animation

## Overview
This repository contains a project that demonstrates animated social media icons using HTML and CSS. The icons include smooth transitions and animations to enhance user interaction and visual appeal.

## Features
- **Animated Icons**: Smooth and visually appealing animations for social media icons.
- **Responsive Design**: Icons adjust gracefully on different screen sizes.
- **Pure CSS**: Utilizes CSS for animations without relying on external libraries.

## Technologies Used
- **HTML**: Structure of the webpage.
- **CSS**: Styling and animations for the icons.

## Getting Started
To get a local copy up and running, follow these simple steps.

### Prerequisites
Make sure you have the following installed:
- A web browser
- Git

### Installation
1. Clone the repository:
    ```sh
    git clone https://github.com/mihaiapostol14/Social-Media-Icon-Animation.git
    ```

2. Navigate to the project directory:
    ```sh
    cd Social-Media-Icon-Animation
    ```

3. Open `index.html` in your web browser to view the animated icons:
    ```sh
    open index.html
    ```

## Usage
Open the `index.html` file in any web browser to see the animated social media icons in action. You can customize the icons and animations by modifying the CSS file.
